# Portal Questões Concurso

Aplicação de questões de concurso com funcionalidades de:
- Resolução de questões
- Filtros por categoria, banca, dificuldade e texto
- Estatísticas de desempenho (incluindo usuários não logados)
- Área administrativa
- Sistema de simulados

## Tecnologias

- React + Vite
- TypeScript
- Node.js + Express (Server)
- SQLite + Drizzle ORM
- Tailwind CSS
- TRPC

## Como rodar

1. Instale as dependências:
```bash
npm install
```

2. Rode o servidor de desenvolvimento:
```bash
npm run dev
```

3. Acesse `http://localhost:3000`
